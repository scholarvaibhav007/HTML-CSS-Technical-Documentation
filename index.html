Examples of JavaScript Code:

Here are five basic JavaScript examples that demonstrate various fundamental concepts:

  1. Hello World
    2. console.log("Hello, World!");

    This simple code prints "Hello, World!" to the console.

  2. Variables
    3. let name = "John"; let age = 30; console.log("Name: " + name); console.log("Age: " + age);

    This code declares two variables, name and age, and prints their values.

  3. Functions
    4. function greet(name) { return "Hello, " + name + "!"; } console.log(greet("Alice"));

    This code defines a function greet that takes a name as an argument and returns a greeting message.

  4. Conditionals
    5. let number = 10; if (number > 5) { console.log("The number is greater than 5."); } else { console.log("The number is 5 or less."); }

    This code uses an if-else statement to check if a number is greater than 5 and prints a message accordingly.

  5. Loops
    6. for (let i = 1; i <= 5; i++) { console.log("Count: " + i); }

    This code uses a for loop to print the numbers from 1 to 5.
